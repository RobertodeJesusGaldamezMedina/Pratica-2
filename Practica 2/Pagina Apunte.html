Bienvenido a mi página HTML


HTML es el lenguaje con el que se escribe las páginas web. Las páginas web pueden ser vistas por el usuario mediante un tipo de aplicación llamada navegador

Este lenguaje nos permite aglutinar textos, sonidos e imagenes y combinarlos a nuestro gusto

El HTML nos permite la introducción de referencias a otra páginas por medio de los enlaces hipertexto
